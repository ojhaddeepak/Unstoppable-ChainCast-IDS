# ChainCast-IDS

[![Netlify Status](https://api.netlify.com/api/v1/badges/ec825bf6-ae35-49c0-8439-2b0b4fbd3a4a/deploy-status)](https://app.netlify.com/sites/chaincast-ids/overview)

ChainCast-IDS is a blockchain stability forecasting and intrusion detection platform.
It applies cyber-security IDS techniques to detect abnormal blockchain behavior and predict instability.

## Features
- Live blockchain monitoring
- Gas congestion forecast
- Stability score
- IDS-based attack detection
- On-chain incident logging

## Tech Stack
- Next.js
- Node.js
- Ethers.js
- Solidity

## Run
Backend:
```bash
cd backend
npm install
node server.js
```

Frontend:
```bash
cd frontend
npm install
npm run dev
```
