# ChainCast-IDS

ChainCast-IDS is a blockchain stability forecasting and intrusion detection platform.
It applies cyber-security IDS techniques to detect abnormal blockchain behavior and predict instability.

## Features
- Live blockchain monitoring
- Gas congestion forecast
- Stability score
- IDS-based attack detection
- On-chain incident logging

## Tech Stack
- Next.js
- Node.js
- Ethers.js
- Solidity

## Run
Backend:
```bash
cd backend
npm install
node server.js
```

Frontend:
```bash
cd frontend
npm install
npm run dev
```
